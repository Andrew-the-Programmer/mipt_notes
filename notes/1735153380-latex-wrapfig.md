# latex wrapfig
https://www.overleaf.com/learn/latex/Wrapping_text_around_figures

```tex
\begin{wrapfigure}{r}{0.5\textwidth}
    \centering
    \includegraphics[width=0.48\textwidth]{birds}
    \caption{Birds}
\end{wrapfigure}
```
