# Фундаментальная последовательность
$(X, \rho)$ - [[1728837879-метрическое-пространство|метрическое пространство]]
$\set{x_n}_{n\in\NN} \subseteq X$
$$
\Gather{
\forall \varepsilon > 0 \exists N : \forall n,m > N \hthen \rho(x_n, x_m) < \varepsilon \\
(\textbf{условие Коши})
}
$$
