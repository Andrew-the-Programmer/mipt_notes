# Первый интеграл \[движения]
# Definition 1
$x = F(x)$

$\phi(x)$ - первый интеграл, если
$$
\phi(x(t)) = C \phi(x(t_0))
$$
# Definition 2
$$
\Gather{
\Cases{
\dot{x} = F(x,t) \\
x(t_0) = x_0
},\quad 
x \in \RR[n],\quad x = x(x_0, t_0, t)
}
$$
$G(x,t) -$ первый интеграл, если
$$
G(x(x_0, t_0), t) = G_0 \equiv const
$$
# Критерий первого интеграла
$$
\dot{{G}_{F}} = \dv{t}{G} + \veccross{\grad{G}}{\vec{F}} = 0
$$
