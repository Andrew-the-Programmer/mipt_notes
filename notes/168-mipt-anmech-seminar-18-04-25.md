# MIPT anmech seminar 18-04-25

# Уравнение Гамильтона-Якоби

Свободные канонические преобразования.

$$
\Cases{
\pdv{q_i}{S} = c p_i \\
\pdv{\tilde{q_i}}{S} = -\tilde{p_i} \\
\tilde{H} = \pdv{t}{S} + c H
}
$$

$\tilde{H} = 0,\quad c = 1 \implies$
Уравнение Гамильтона-Якоби:

$$
\pdv{t}{S} + H(t,q,\pdv{q}{S}) = 0
$$
