# Независимые события
# Definition
[[943-условная-вероятность|условная вероятность]]

События A и B называются **_независимыми_**, если

$$
P(A \mid B) = P(A)
$$

$\implies P(B \mid A) = P(B)$

# Независимость не транзитивна
# Claim
$$
P(AB) = P(A) P(B)
$$
# Fact
$P(A) = 1, 0 \implies$
