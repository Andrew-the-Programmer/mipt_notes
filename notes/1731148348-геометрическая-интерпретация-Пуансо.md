# геометрическая интерпретация Пуансо
$$
\alignleft{
f(r) = r^T J r - 1 = 0 \\
r = \lambda \omega \\
\lambda = \frac{1}{\sqrt{2T}} \\
f(r) = \lambda \omega^T J \omega - 1 = 0 \\
\grad{f} = 2 J \vec{r} \\
\grad f = \frac{2}{\sqrt{2T}} J \omega = \frac{2 K}{\sqrt{2T}} \equiv const
}
$$
